# notify.js
一个简单的顶部提示组件

#### 使用

1. 引用 notify.js

```
<script src="notify.js"></script>
```

2. 方法

```
notify.alert(1, '成功提示', 1.5);
notify.alert(2, '警告提示', 1.5);
notify.alert(3, '危险提示', 1.5);
notify.alert(4, '正常提示', 1.5);
```
其中 1，2，3，4代表四种提示的类型，1==success, 2==warn, 3==danger, 4==primary，1.5表示提示多久后消失
